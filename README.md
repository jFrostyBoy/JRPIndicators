## JRPIndicators

**Ядро**: Paper / Spigot  
**Версия**: 1.16.5 - 1.21.11  
**Java**: 16+  
**Зависимость:** PlaceholderAPI  

Плагин добавляет реалистичные игровые индикаторы времени, погоды, календаря, сезонов и направления взгляда игрока.  
Не добавляет никаких механик, по типу смены цвета биомов под сезон и тд, а лишь служит визуально вымодимой информацией.  
Идеально подходит для ролевых серверов, где важна атмосфера и погружение.

### Особенности
- Игровой календарь с 12 месяцами по 28 дней (настраивается)
- Времена года: Весна, Лето, Осень, Зима
- Время суток: Утро, День, Вечер, Ночь
- Погода: Ясно, Дождь, Снег, Гроза
- Направление взгляда игрока
- Автоматические приветствия при смене:
    - Времени суток
    - Погоды
    - Сезона (года)
- Полная кастомизация через `config.yml`

### Установка

1. Поместите файл `JRPIndicators.jar` в папку `plugins/`.
2. Перезапустите сервер.
3. (Опционально) Отредактируйте `plugins/JRPIndicators/config.yml` под свои нужды.
4. Выполните `/rpreload` для применения изменений.

### Админ-команда

| Команда     | Описание                           | Право доступа         |
|-------------|------------------------------------|-----------------------|
| `/rpreload` | Перезагрузить конфигурацию плагина | `jrpindicators.admin` |

## Права (Permissions)
- `jrpindicators.admin` — доступ к команде перезагрузки (`/rpreload`)

## Плейсхолдеры
> Можно использовать где душе угодно, а так же почти в любых плагинах,
> если они имеют поддержку PlaceholderAPI 

| Плейсхолдер              | Описание                              | Пример вывода |
|--------------------------|---------------------------------------|---------------|
| `%jrpi_day_type%`        | Время суток                           | Утро          |
| `%jrpi_time%`            | Игровое время (по формату из конфига) | 14:35         |
| `%jrpi_weather%`         | Текущая погода                        | Ясно          |
| `%jrpi_day%`             | День месяца (1–28)                    | 15            |
| `%jrpi_month%`           | Номер месяца (1–12)                   | 6             |
| `%jrpi_month_name%`      | Название месяца                       | Июнь          |
| `%jrpi_weekday%`         | Номер дня недели (1–7)                | 3             |
| `%jrpi_weekday_name%`    | Название дня недели                   | Среда         |
| `%jrpi_season%`          | Текущий сезон                         | Лето          |
| `%jrpi_direction%`       | Полное направление взгляда            | Юго-Запад     |
| `%jrpi_direction_short%` | Короткое направление                  | ЮЗ            |

> Все текстовые значения берутся из конфига и поддерживают цветовые коды `&`.
